# Banking System Simulator

A robust banking system simulator built with Python, designed to model real-world banking workflows through secure authentication, account management, transaction processing, balance tracking, and persistent data storage.

## Overview

This project simulates the core functionalities of a modern banking system. It allows users to create and manage accounts, perform transactions, track balances, and maintain transaction records while demonstrating object-oriented programming and software design principles.

## Features

- User Registration and Authentication
- Account Creation and Management
- Deposit Operations
- Withdrawal Operations
- Balance Inquiry
- Transaction History Tracking
- Secure User Access
- Persistent Data Storage
- Object-Oriented Design
- Error Handling and Input Validation

## Technologies Used

## Installation

Clone the repository:

```bash
git clone https://github.com/HITESHSEERVI-123/Banking-System-Simulator.git
```

Move into the project directory:

```bash
cd Banking-System-Simulator
```

Run the application:

```bash
python main.py
```
